2. CineBook: Terminal-Based Movie Reservation System GitHub

CineBook is a Python-based terminal reservation system that allows users to book seats for movies and enables staff to add new movies to the theatre list. Built with a creative UI using ASCII graphics and colorama, it simulates a real cinema booking flow for both customers and administrators.

• Developed in high school as one of my first large Python projects.
• Interactive seat layout with real-time visual booking representation.
• Staff login functionality for dynamic movie management.
• ASCII-based design for a unique visual experience inside terminal.
• No external frameworks used — built purely with Python standard modules and logic.
